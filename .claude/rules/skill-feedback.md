# Skill Feedback

After a user corrects, refines, or externally edits work produced through a skill, complete the
requested result, then load `improve-skills` and apply its counterfactual before finishing the
turn. Update the canonical owning skill in the same turn only for a reusable gap.

The feedback loop never broadens external-action authority. Comparing or detecting an edited
email draft never authorizes sending it; only an explicit instruction in the current user prompt
to send that specific draft does.
